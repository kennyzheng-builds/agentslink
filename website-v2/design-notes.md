# AgentsLink 官网设计说明

## 设计哲学

### 核心理念
AgentsLink 是一个让 AI Agent 之间直接协作的工具。设计目标是传达**温暖、信任、简洁**的感觉 —— 像介绍两个朋友认识一样自然。

### 避免的 AI 设计套路
- ❌ 不使用 cyan/purple 渐变（典型的 AI 产品配色）
- ❌ 不使用圆角卡片堆叠（过度使用的布局模式）
- ❌ 不使用玻璃拟态（Glassmorphism）效果
- ❌ 不使用发光边框或霓虹效果
- ❌ 不使用大图标 + 标题 + 描述的三段式卡片

---

## 视觉设计

### 配色方案

采用**温暖、大地色系**，营造人文、可信的氛围：

| 颜色 | 色值 | 用途 |
|------|------|------|
| Cream | `#FDFBF7` | 主背景色，温暖米白 |
| Warm | `#F7F3ED` | 次级背景，区块分隔 |
| Sand | `#EDE6DC` | 卡片背景、hover 状态 |
| Ink Primary | `#1C1917` | 主文字，近黑色 |
| Ink Secondary | `#4A4540` | 次级文字 |
| Ink Muted | `#78726A` | 辅助文字 |
| Terracotta | `#B85C3F` | 主强调色，陶土红 |
| Sage | `#6B7B6A` | 次强调色，鼠尾草绿 |
| Ochre | `#C9A66B` | 点缀色，赭石黄 |

### 字体搭配

- **标题**: Crimson Pro (衬线体) — 优雅、人文、有温度
- **正文**: DM Sans (无衬线体) — 现代、清晰、易读

这种搭配避免了科技产品常用的纯无衬线字体，增添了人文气息。

### 布局原则

1. **大量留白** — 让内容呼吸
2. **不对称平衡** — Hero 区域文字偏左，视觉元素偏右
3. **微妙的边框** — 使用渐变边框分隔区块，而非生硬的线条
4. **一致的圆角** — 统一使用 4px-12px 的小圆角，保持克制

## 动效设计

### 入场动画

- **Hero 区域**: 文字依次从下方淡入，营造渐进揭示感
- **卡片**: 滚动进入视口时淡入 + 上移
- **时序**: 使用 `cubic-bezier(0.16, 1, 0.3, 1)` (ease-out-expo) 实现自然的减速效果

### 微交互

| 元素 | 交互 | 效果 |
|------|------|------|
| Logo | Hover | 两个圆点相互靠近 |
| 导航链接 | Hover | 下划线从左向右展开 |
| 主按钮 | Hover | 背景色从下方滑入，轻微上浮 |
| 步骤卡片 | Hover | 上浮 4px，阴影加深 |
| 功能图标 | Hover | 放大 10%，背景变色 |

### 装饰动画

- **Hero 背景**: 两个渐变圆缓慢浮动，增加空间感
- **连接可视化**: 
  - 中心点脉冲发光
  - 粒子沿连接线流动

## 响应式设计

### 断点

- **Desktop**: > 1024px — 完整布局
- **Tablet**: 768px - 1024px — 隐藏 Hero 视觉元素，单列步骤
- **Mobile**: < 768px — 全宽按钮，堆叠布局

### 移动端适配

- 导航链接隐藏（保留 Logo）
- 按钮全宽显示
- 减少内边距（32px → 20px）
- 卡片全宽显示

---

## 可访问性

- 支持 `prefers-reduced-motion` 媒体查询，禁用动画
- 所有交互元素有清晰的 focus 状态
- 颜色对比度符合 WCAG AA 标准
- 语义化 HTML 结构

## 与初稿的改进对比

| 方面 | 初稿 | V2 |
|------|------|-----|
| 配色 | 偏冷的灰色调 | 温暖的大地色系 |
| 字体 | Playfair + Source Sans | Crimson Pro + DM Sans |
| Hero 视觉 | 简单圆点连接 | 动态粒子 + 脉冲效果 |
| 卡片设计 | 扁平简洁 | 微阴影 + hover 上浮 |
| 动效 | 基本淡入 | 精心编排的入场序列 |
| 整体感觉 | 功能导向 | 人文 + 精致 |

## 技术实现

- **纯 HTML + CSS**: 无框架依赖
- **原生 JS**: 仅用于滚动监听和交互动画
- **CSS 变量**: 统一管理系统颜色、缓动函数
- **现代 CSS**: 使用 clamp() 实现流体排版，grid 布局
