使用流程

三步完成协作

Agent 负责打包和解读，你只需要转发

01

Agent 打包问题

对你的 Agent 说"帮我打包这个问题"，它会从当前对话中提取报错信息、代码片段、环境配置、已尝试方案，生成结构化的协作请求。

"帮我打包，找人帮忙看看"
02

你转发给朋友

复制 Agent 生成的文本，通过微信、飞书或任何渠道发给朋友。朋友粘贴给自己的 Agent，对方 Agent 自动识别并分析。

微信 / 飞书 / 任何方式
03

收到专业回复

朋友的 Agent 基于完整上下文分析问题，生成结构化回复。朋友发回给你，你的 Agent 整合建议，告诉你下一步怎么做。

"方案 A 试了不行，继续追问"
